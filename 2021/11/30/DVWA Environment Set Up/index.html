前言

本文章为web教学前的本地环境平台搭建教学，为之后web安全的教学做铺垫，所要用到的环境和软件可以到我个人云盘下载。本篇数据库相关配置有疑惑的可以移步《mysql数据库操作基础》查看相关命令。本篇环境搭建步骤如有问题或在安装过程有出现未知错误，可以联系我，在博客的主页有我个人联系方式。

Basic

本次搭建的环境为dvwa平台，所要用到的软件为phpstudy。

配置dvwa平台

1.将dvwa平台压缩包解压到phpstudy目录下的 WWW 文件夹中

2.进入dvwa平台目录修改配置文件（我的文件路径：E:\phpstudy\phpstudy_pro\WWW\DVWA-master\config），找到 config.inc.php.dist 文件进行复制改名为 config.inc.php打开文件进行配置修改

配置phpstudy

Apache安装配置操作步骤

1.先将phpstudy 安装到本地目录

2.打开phpstudy 先点击设置 –> 系统设置 –> 端口检测，输入80 和 443 （后面dvwa配置，web配置服务要用查看是否被占用，方便之后修改）。

3.如若发现端口被占用，可以更改Apache 的配置，点击首页 –> 找到Apache2.4.39点击配置 –> 找到启动端口修改一个未被占用的端口（例如8080、8088等），如果默认端口（http 80 https 443）未被占用则无需修改。
